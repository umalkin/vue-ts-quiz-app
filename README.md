# Simple Quiz with Vue 3 and TypeScript

A simple quiz web app using Vue 3 with Vite and TypeScript.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
